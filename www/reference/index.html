  • RiGrammar

    A probabilistic context-free grammar with literary extensions designed for text-generation 
     
		rg = new RiGrammar("mygrammar.g");
		System.out.println(rg.expand());
    RiTa grammar files are JSON-formatted text files that follow the format below: 
       

		  "<start>": "<rule1> | <rule2> | <rule3>"
	
		  "<rule2>": "terminal1 |  terminal2 | <rule1>"
		
		   .

  • RiLexicon

    The core 'dictionary' (or lexicon) for the RiTa tools.

  • RiMarkov

    Performs text generation via Markov chains (aka n-grams) with options to process single characters, words, sentences, and arbitrary regular expressions.

  • RiString

    The basic text container object, implementing a variety of additional functionality atop the javascript string object

  • RiTa

    A collection of static variables and functions for the RiTa library

  • RiTaEvent

    A simple wrapper for event-based callbacks

  • RiText

    A text display object for the RiTa tools.
