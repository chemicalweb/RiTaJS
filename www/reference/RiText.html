Class RiText

Examples 
import rita.*;
     
void setup() {
   // start in the upper left corner
   RiText rt = new RiText(this, "hello", 0, 10);
   
   // measure the width of the text
   float tw = rt.textWidth();
  
   // move to the lower right over 2 sec
   rt.moveTo(width-tw, height, 2);    
}

void draw() {
   // draw the bg every frame
   background(255);
}
Description A text display object for the RiTa tools. Wraps an instance of RiString to provide utility methods for typography, text effects, animation, etc.

Uses either the native canvas renderer or the Processing renderer (when included)
