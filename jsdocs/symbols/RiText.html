Class RiText

RiTa's text display object. Wraps an instance of RiString to provide utility methods for typography, text effects, animation, etc. Uses either the native canvas renderer or the Processing renderer (when included)
Defined in: rita.js.

<static> {array} RiText.picked(x, y)
Returns all RiTexts that contain the point x,y or null if none do.

Note: this will return an array even if only one item is picked, therefore, you should generally use it as follows: 

  var rts = RiText.getPicked(mx, my);
  if (rts.length) {
     rts[0].doSomething();
  }
